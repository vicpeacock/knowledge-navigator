# Guida Registrazione Video Kaggle - Screen Recording + Voiceover Separati

## Workflow Consigliato

### Step 1: Registrare Screen Recording (senza audio)

#### Opzione A: Screen Recording macOS senza microfono

1. Premi `Cmd + Shift + 5`
2. Seleziona **"Record Entire Screen"** o **"Record Selected Portion"**
3. Clicca **"Options"** ‚Üí **"Microphone: None"** (o disattiva il microfono nelle impostazioni di sistema)
4. Clicca **"Record"**
5. Esegui tutte le interazioni seguendo lo script
6. Ferma la registrazione: clicca l'icona nella barra dei menu o `Cmd + Control + Esc`

#### Opzione B: QuickTime Player senza audio

1. Apri QuickTime Player
2. File > New Screen Recording
3. Clicca la freccia accanto al pulsante di registrazione
4. **NON selezionare nessun microfono** (o scegli "None")
5. Clicca "Record"
6. Esegui le interazioni
7. Ferma la registrazione

**Output**: Video senza audio (solo screen recording)

---

### Step 2: Preparare il Voiceover

Prima di registrare il voiceover:

1. **Prepara lo script**: Usa `scripts/kaggle-video-script.md`
2. **Leggi ad alta voce**: Pratica il timing per ogni sezione
3. **Usa un microfono**: Migliore qualit√† possibile (anche AirPods vanno bene)
4. **Ambiente silenzioso**: Riduci rumori di fondo

---

### Step 3: Editing con iMovie (Aggiungere Voiceover)

#### Importare il Video

1. Apri **iMovie** (Applications)
2. Clicca **"Create New"** ‚Üí **"Movie"**
3. Trascina il video screen recording nella timeline
4. Il video apparir√† nel progetto

#### Registrare Voiceover Direttamente in iMovie

1. **Posiziona il playhead** all'inizio del video (o nella sezione dove vuoi iniziare il voiceover)
2. Clicca l'icona **microfono** sopra la timeline (o `V` sulla tastiera)
3. Si aprir√† una finestra di registrazione voiceover
4. **Controlli disponibili**:
   - Pulsante "Record" per iniziare
   - Pulsante "Stop" per fermare
   - Visualizzazione livello audio
   - Controllo volume

5. **Registra il voiceover**:
   - Leggi lo script guardando il video
   - Puoi registrare tutto in una volta o sezione per sezione
   - Premi "Record", poi "Stop" quando finisci una sezione
   - Ripeti per ogni sezione

6. **Registrare sezioni separate**:
   - Posiziona il playhead dove vuoi iniziare la prossima sezione
   - Clicca di nuovo il microfono
   - Registra la sezione successiva
   - Le registrazioni saranno automaticamente sincronizzate con il video

#### Editing del Voiceover

1. **Seleziona il clip audio** del voiceover nella timeline
2. **Aggiusta il volume**: Trascina il fade in/out o usa il controllo volume
3. **Sincronizzazione**: Trascina il clip audio per allinearlo con le azioni nel video
4. **Rimuovere errori**: Se hai sbagliato una parte, seleziona quella parte e premi `Delete`, poi registra di nuovo quella sezione

#### Suggerimenti per iMovie

- **Timing**: Registra guardando il video per sincronizzare meglio le spiegazioni
- **Pause**: Puoi inserire pause nel voiceover cliccando "Stop" e spostando il playhead
- **Sovrapposizioni**: Puoi avere pi√π clip audio se necessario
- **Musica di sottofondo** (opzionale): Aggiungi musica soft a volume basso da iMovie Library

---

### Step 4: Esportare il Video

1. Clicca **"Share"** (icona in alto a destra) o `Cmd + E`
2. Seleziona **"File"**
3. **Impostazioni**:
   - **Resolution**: 1080p (1920x1080) o 720p (1280x720) per YouTube
   - **Quality**: High
   - **Compress**: Better Quality
4. Clicca **"Next"**
5. Scegli dove salvare (Desktop o Downloads)
6. Clicca **"Save"**

**Tempo di export**: 2-5 minuti a seconda della lunghezza

## Workflow Alternativo: Voiceover Esterno + iMovie

Se preferisci registrare il voiceover separatamente (pi√π controllo):

### Registrare Voiceover con QuickTime o Voice Memos

1. **QuickTime Player**:
   - File > New Audio Recording
   - Registra tutto lo script
   - Salva come file audio (.m4a)

2. **Voice Memos** (iPhone/Mac):
   - Registra lo script completo
   - Condividi il file sul Mac

### Importare in iMovie

1. Trascina il file audio nella timeline sotto il video
2. Sincronizza manualmente trascinando l'audio
3. Taglia l'audio per rimuovere pause lunghe o errori
4. Aggiusta il volume
